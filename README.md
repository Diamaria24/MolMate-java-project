Chemistry learners often struggle with quickly accessing reliable information such
as chemical name formula conversions, molar masses, and composition data. Relying on
textbooks, manual calculations, or scattered online searches can be time-consuming and
prone to errors, especially during study sessions, lab work, or examinations. To address
this problem, we propose the MolMate, a Java based desktop application that provides a
fast, user-friendly solution for common chemistry tasks. The application supports both
console interaction for simplicity and an enhanced GUI interface for wider usability. The
system is designed to perform: formula ↔ name conversion using HashMaps and
optional PubChem API support, molar-mass calculation for compounds using atomic
weights, composition percentage breakdown of elements in a compound, periodic-table
lookups for quick access to element properties, quiz-mode with progress tracking to make
the tool interactive for learning, and reference notes plus export options for saving results.
The software integrates JavaFX for GUI, HashMap data structures, JSON/CSV storage
for offline lookups, and HTTP client libraries for optional real-time data retrieval— this
combination ensures both offline efficiency and scalability. By merging educational
utility with interactivity, ChemConvert Assistant not only simplifies routine chemical
tasks but also promotes active learning. It demonstrates how Java programming, data
structures, API integration, and GUI development can be applied to create a practical,
extendable tool for students, teachers, and researchers in chemistry.
